# 🏓 Tennis Game

這是一款使用 HTML5 Canvas 製作的小遊戲，玩法類似經典 Pong：

- 玩家 vs 電腦對戰

- 積分制度，先達到目標分數者獲勝

- 可以無限重新挑戰

直接點擊以下連結開始遊玩：
👉 <https://yudododo.github.io/tennis_game/>

## 🛠 使用技術

HTML5 Canvas：繪製遊戲畫面

JavaScript (ES6)：遊戲邏輯與互動
